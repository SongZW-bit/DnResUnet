# Results Folder Notes

The `independent_resampling_v3` folder contains the latest independent evaluation outputs corresponding to the manuscript:

- `independent_resampling_summary.csv`
  Summary table with mean, standard deviation, and confidence-interval statistics.
- `independent_resampling_raw_metrics.csv`
  Per-sample metrics exported by the evaluation workflow.
- `independent_resampling_report.json`
  Compact machine-readable summary of the benchmark run.
- `field_review_template.csv`
  Template for qualitative review when no clean field-data labels are available.
- `*.pdf`
  Grouped figures used for manuscript tables/figures and visual comparison.
